### Что такое реактивность во Vue?

**Хороший ответ (для HR):**
> Vue автоматически отслеживает зависимости данных и обновляет DOM при их изменении. В Vue 3 это работает через `ref` и `reactive` на основе JavaScript Proxy. Я определяю реактивные данные и Vue сам решает, какой компонент нужно перерисовать.

---

### Что такое хуки в React?

**Хороший ответ (для HR):**
> Хуки — это функции, которые позволяют использовать состояние и жизненный цикл в функциональных компонентах. Основные: `useState` для состояния, `useEffect` для сайд-эффектов, `useContext` для контекста. Использую также кастомные хуки для переиспользования логики.

---

### Что такое TypeScript и зачем он нужен?

**Хороший ответ:**
> TypeScript — это JavaScript с типами. Даёт: автодополнение в IDE, раннее обнаружение ошибок на этапе компиляции, лучшую читаемость кода в команде. На больших проектах существенно снижает количество runtime-ошибок.
