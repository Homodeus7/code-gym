## Советы перед звонком

1. **Изучи компанию** — сайт, продукт, вакансия, стек
2. **Подготовь цифры** — зарплатные ожидания, даты опыта
3. **Иметь 2–3 истории** по методу STAR (Situation, Task, Action, Result)
4. **Тихое место** — камера включена, нет фоновых звуков
5. **Вопросы к HR** — минимум 2–3 готовых вопроса

## Метод STAR

Используй для вопросов вида «Расскажи случай, когда...»

- **S**ituation — контекст, что происходило
- **T**ask — твоя задача / роль
- **A**ction — что конкретно сделал
- **R**esult — результат (желательно с метриками)
